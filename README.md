# Expansion-ScoreboardObjectives

```
%objective_score_{<obj-name>}%
%objective_score_{<obj-name>}_{<otherEntry>}%
%objective_scorep_{<obj-name>}% - Exactly the same as %objective_score_<obj-name>%
%objective_scorep_{<obj-name>}_{<otherPlayer>}% = will check if the optional parameter <otherPlayer> is a player, if not returns PNF
%objective_displayname_<obj-name>% - Will get the DisplayName of the given Objective, NOTE: the obj-name is case sensitive
```
